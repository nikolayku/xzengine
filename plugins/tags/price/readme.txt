<p style="text-align: justify;">
<b>Общяя информация</b><br>
Плагин для создания настраиваемых тегов. Код с тегами разполагается в папке 'tags' каждый в разном файле<br>
Имя файла и есть тэг 
К примеру код для li.ru располагается в файле liveinternet,
при этом тег для вывода счётчика будет вида {liveinternet} и может быть расположен в любом месте на странице<br><br>

<b>Как использовать плагин:</b> <br>
В поле 'имя тега' указываем имя будущего тега в поле 'Код тега' вставляем любую текстовыю информацию,
(код счётчика, код adcense, или просто текст)<br>
Жмём по кнопке 'создать', если небыло ошибок создания 
то тег для вставки отобразится снизу формы ввода настроек.  Этот тег можно вставить в любое место 
страницы и плагин отобразит вместо тега его вставленный код. Разрешены основные теги { sitepath} и другие

</p>
