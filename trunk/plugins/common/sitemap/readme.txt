Далее прведена статья о том что такое sitemap и зачем он надо 

Что это такое
Файл sitemap.xml это специальный файл, служащий для описания страниц вашего сайта. Это файл, разработанный поисковой 
системой Google для того чтобы знать какие страницы сайта подлежат индексации. Иными словами те страницы, которые 
есть на сайте. Это ускоряет индексацию сайта (так как все страницы находятся в одном файле) и нагрузку на сервер 
(так как проще запросить один файл, нежели сканить все страницы начиная, с главной). Многие поисковые системы 
сегодня используют файл sitemap для более быстрой индексации.

Формат файла
Файл sitemap.xml – это обычный файл формата xml. В общем виде файл sitemap.xml состоит из блоков типа

URL
PRIORITY


Здесь URL это полный путь к странице сайта, а PRIORITY это дробное число от 0 до 1, показывающее приоритет 
страницы. То есть приоритет страницы для поисковой системы. Но современные поисковые системы сами выставляют 
приоритет в зависимости от их загруженности. Так что этот параметр это не более чем предложение поисковой 
системе, но, тем не менее не стоит им пренебрегать. Стандартное значение 1.0 (высший приоритет ) для страницы 
первого уровня, 0.8-0.5 для страниц 2го уровня вложенности  ниже 0.4 для страниц 3го уровня.

Плагин для xzengine генерации sitemap.xml
В комплект поставки xzengine входит плагин для генерации sitemap.xml. Если сервер поддерживает 
поддержку .htaccess то запустить этот плагин можно набрав http://имя_вашего_сайта/sitemap.xml Если сервер 
не поддерживает файл .htaccess .(что характерно для бесплатных хостингов) то можно запустить сам плагин. 
Для этого надо выполнить http://имя_вашего_сайта/plugins/sitemap/index.php
Также можно настроить плагин для того чтобы он сохранял файл sitemap.xml в ту директорию, которую 
вам необходимо. Для этого необходимо открыть файл index.php лежащий в директории плагина 
(/plugins/sitemap/index.php) любым текстовым редактором найти строку $use_htacess = true; и написать  
$use_htacess = false; строка $saveToPath = '../../sitemap.xml'; указывает на то в какой директории 
сохранять требуемый файл. На этот файл должны быть выставлены права 777 (полный доступ). 
Недостаток этого подхода в том, что надо постоянно запускать файл /plugins/sitemap/index.php 
для постоянной генерации файла sitemap.xml
