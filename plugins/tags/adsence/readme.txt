Плагин для размещения рекламных блоков goole adsence
в директории banners расположены файлы они представляют собой теги
то есть если файл называется 'файл.txt' то тег будит {файл}
Можно добавлять туда своих файлов которые автоматом будут преобразованы в теги.

Пример использования тегов, можно найти в скине xzengine_ru_plugins
расположенном в папке skin
