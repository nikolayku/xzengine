<p style="text-align: justify;">
<b>Общяя информация</b><br>
Плагин для счётчиков. Код со счётчиками разполагается в папке counters каждый в разном файле<br>
Имя файла и есть тэг 
К примеру код для li.ru располагается в файле liveinternet
тогда тег для вывода счётчика будет вида {liveinternet} и может быть расположен в любом месте на странице<br><br>

<b>Как использовать плагин:</b> <br>
В поле 'имя счётчика' указываем имя будущего счётчика(или тега) в поле 'Код счётчика' копируем код счётчика<br>
его можно взять при регистриции нового сайта. Жмём по кнопке 'создать', если небыло ошибок создания 
то тег для вставки отобразится снизу формы ввода настроек счётчика.  Этот тег можно вставить в любое место 
страницы и плагин отобразит код привязанного к тегу счётчика.

</p>
